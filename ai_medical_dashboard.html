ℹ About This System
AI-powered clinical decision support for radiological diagnosis
🚀 Developed as an AI-powered healthcare diagnostic solution using deep learning and GradCAM explainability.
🧠
Project Overview
An intelligent healthcare application designed to analyze chest X-ray images and detect lung diseases such as Pneumonia and Tuberculosis using deep learning.
⚙
Key Features
AI-based X-ray detection · Real GradCAM heatmap visualization · Clinical interpretation · Automated PDF reports · Claude AI Doctor chatbot · Precaution planning.
🔬
Technologies
Python · TensorFlow/Keras · OpenCV · GradCAM · ReportLab · Chart.js · FastAPI · Anthropic Claude API.
🏥
Real-World Impact
Assists healthcare professionals in early disease detection, reduces diagnostic time, and provides clinical decision support — especially in areas with limited radiologist access.
🚀
How It Works
1. Upload chest X-ray · 2. FastAPI /api/predict runs model · 3. Real GradCAM via /api/gradcam · 4. Clinical insights generated · 5. PDF report from /api/report/pdf.
🎯
Model Performance
94.2% overall accuracy · 92% precision · 91% recall · 91.5% F1 score across Normal, Pneumonia, and Tuberculosis classification.
